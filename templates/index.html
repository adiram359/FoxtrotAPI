Foxtrot API

Hey there! Thanks for checking out the Foxtrot API. Foxtrot was one of my favorite comics I read when I was younger, so I decided to build this API. FoxtrotAPI can can fetch comics strips from a database of 400+ stips! Right now, there are two ways to fetch a comic strip:
1. Fetch a random comic strip:

var result = await fetch( https://foxtrot.io/random )

response:
/* { "id_": 1, "author": "Bill Amend", "title": "Iguanoman", "img_src": "iguanoman.png" "date": Month.Date.Year "success": true} */
2. Fetch a strip with a specfic title:

var result = await fetch( https://foxtrot.io/title/strip_title ) //replace strip_title and use "." in place of space"

response:
/* { "id_": 1, "author": "Bill Amend", "title": "Iguanoman", "img_src": "iguanoman.png" "date": Month.Date.Year "success": true} */
3. Fetch a strip by the date it was published:

var result = await fetch( https://foxtrot.io/date/Month.Date.Year ) //format is MM.DD.YYYY

response:
/* { "id_": 1, "author": "Bill Amend", "title": "Iguanoman", "img_src": "iguanoman.png" "date": Month.Date.Year "success": true} */
