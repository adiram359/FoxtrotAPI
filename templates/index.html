Foxtrot API

Hey there! Thanks for checking out the Foxtrot API. Foxtrot was one of my favorite comics I read when I was younger, so I decided to build this API. FoxtrotAPI can can fetch comics strips from a database of 400+ stips! Right now, there are three ways to fetch a comic strip, but if you're not interested in the API and just want to see the API in action click "Fetch Another Comic!" down below.
1. Fetch a random comic strip:

var result = await fetch( https://foxtrotapi.herokuapp.com/random )

2. Fetch a strip with a specfic title:

var result = await fetch( https://foxtrotapi.herokuapp.com/title/strip_title )

//replace strip_title and use "." in place of spaces"

3. Fetch a strip by the date it was published:

var result = await fetch( https://foxtrotapi.herokuapp.com/date/Month.Date.Year )

//replace strip_title and use "." in place of spaces"
